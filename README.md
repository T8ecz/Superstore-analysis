# Superstore Sales & Profit Analysis

## Project Overview

This project analyzes Superstore sales data to identify key business drivers, profitability trends, customer behavior, and the impact of discounts on company performance.

The analysis was conducted using Python and focuses on transforming raw sales data into actionable business insights and recommendations.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook / VS Code

## Dataset

The dataset contains historical Superstore transactions, including:

* Sales
* Profit
* Discount
* Quantity
* Customer Segment
* Category
* Sub-Category
* Region
* Shipping Mode
* Order Date

## Key Insights

* The West region generated the highest sales and profit among all regions.
* Orders with discounts below 30% generated positive profits, while larger discounts often resulted in losses.
* December was the most profitable month, while 2017 recorded the highest annual profit.
* The Technology category generated the highest profit among all categories.
* Technology products shipped through Standard Class generated the highest sales revenue.
* The Consumer segment generated the highest sales and profit among all customer segments.

## Business Recommendations

* Limit discounts above 30% to improve profitability.
* Continue investing in the West region and the Technology category due to their strong contribution to revenue and profit.
